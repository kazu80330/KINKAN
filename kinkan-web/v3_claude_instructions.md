# KINKAN Web App - 第3フェーズ（UIレイアウトと棒グラフの完全復元）

Claude Codeへ。第2フェーズの実装お疲れ様でした。機能面は揃いましたが、さらに**オリジナル版（元HTML）の美しいUIスタイルとレイアウトを現在のReact版に完全に移植**することが今回の目標です。

ユーザーは「元HTMLの1カラムスタイルや、棒グラフ機能」を大変気に入っています。以下の項目を実装してください。

### 1. 全体レイアウトを「オリジナルHTMLの縦積み・ワンカラム」に近づける
現在の `App.tsx` では `.dashboard-grid` などを使ってカードが横並びになっている可能性がありますが、元の `kinkan.html` は**「画面中央配置・最大幅 680px 程度」の縦積みレイアウト**でした。
- **要件**: PC画面でも横に広がらず、スマホやタブレットのような固定幅（`max-width: 680px` 前後、左右マージン `auto`）で、各カード（ステータス、出退勤、ポモドーロ、休憩）が綺麗に縦に並ぶスタイルにCSS（`index.css`）を適宜修正・整理してください。

### 2. 月次レポートへの「日別勤務時間 棒グラフ（Bar Chart）」の復元
元HTMLのレポート画面の下部には、その月の日ごとの勤務時間を視覚的に確認できる「棒グラフ」が存在し、「⬇ PNG保存」ボタンがありました。
- **要件**: `MonthlyReport.tsx` 内に、日別勤務時間の棒グラフ（Canvas、もしくはHTML+CSSを活用したバーチャート）を実装してください。（元HTMLの `.bar-chart`, `.bar-col`, `.bar-fill` などのCSSやスクリプトの実装を参考にしてください）。
- **要件**: 「⬇ PNG保存」ボタンを追加し、そのグラフ（またはレポート全体）を画像としてダウンロードできる機能（`html2canvas` パッケージ等を利用）を復元・実装してください。

### 3. カード内の微妙な配置の復刻
- ポモドーロの「リセット」「スタート」「スキップ」ボタン群がタイマーのすぐ下に配置されるなど、細かい要素の並び（フレックスボックスの使い方）を旧デザインに限りなく寄せてください。
