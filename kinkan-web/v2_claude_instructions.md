# KINKAN Web App - 追加機能実装指示書 (v2)

Claude Codeへ。KINKAN Webアプリの基本統合とデプロイ設定が成功しました。
しかし、移行元の元HTML（`kinkan.html`）には存在していましたが、現在のReact版で未実装となっている重要な機能がいくつかあります。ユーザーからのリクエストにより、現在のモダンなグラスモーフィズムデザイン（`index.css`）を維持したまま、以下の「旧機能」をReact上に復元・実装してください。

### 全体の方針
- 洗練されたグラスモーフィズムUIを維持してください。
- 既存の `Home` / `Report` タブに機能を追加しても良いですし、情報過多になるようであれば新しく `Settings` または `Break & Tools` といった新しい機能タブ（ページ）を作成しても構いません。最適なUXを考慮して配置してください。

---

## 実装してほしい機能（未実装の旧仕様）

### 1. 手動の「休憩」トラッキング機能 (Break Card)
現在のポモドーロとは別に、「昼休み」や「長時間の離席」を計測・記録するための独立したコントロールが必要です。
- **要件**: 「☕ 休憩開始」および「✓ 休憩終了」ボタンを配置。
- **動作**: 休憩経過時間をカウントアップし、その日の `DailyRecord` の休憩合計時間（`totalBreakSec`）および休憩回数（`breaks`）に加算・永続化する。出退勤の際にも休憩時間が適切に反映されるようにしてください。

### 2. 過去記録の「手動編集 / 削除」モーダル機能
元HTMLでは、レポートのリストをクリックして出退勤時刻を手動で上書きできました。
- **要件**: `MonthlyReport.tsx` のカレンダー行（または専用の編集ボタン）をクリックした際に、モーダルダイアログを表示する。
- **動作**: モーダル内で「出勤時刻」「退勤時刻」「休憩回数」「ポモドーロ数」を書き換え可能にし、保存ボタンでローカルストレージ（およびGoogle Sheets同期）へ反映。また、その日の記録自体を**削除**する機能も含める。

### 3. Excel (.xlsx) データエクスポート機能
- **要件**: 月次レポート画面等に「⬇ .xlsxでエクスポート」のボタンを追加。
- **動作**: `xlsx` パッケージ等を `npm install xlsx` 等で導入し、現在表示している月の `DailyRecord` 群を表計算形式でダウンロードできるようにする。

### 4. 日跨ぎ（オーバーナイト）勤務の対応
- **要件**: 例えば 23:00 に出勤して 翌 02:00 に退勤した場合、勤務時間がマイナスにならず、出勤日の記録として「退勤: 26:00:00」もしくは「+1日 02:00:00」のように適切に計算・表示・保存が行われるロジックを復元する。

### 5. ポモドーロの「出勤時オートスタート」と「サウンド通知」
- **要件**: 
  - オプションとして「出勤ボタンを押した際に、自動的にポモドーロタイマー（作業）をスタートさせる」機能を付与（チェックボックスなどで切り替え可能に）。
  - 各セッション（作業完了・休憩完了）の際に、旧版の AudioContext オシレーター（または Web Audio API）を利用した「サウンド通知機能（チーンという音）」を復元。
