================================================================
  KINKAN - 勤怠管理アプリ  使い方ガイド
================================================================

【ファイル構成】
  kinkan.html     ... アプリ本体
  KINKAN.bat      ... 起動用ファイル（ダブルクリックで起動）
  README.txt      ... このファイル

  ※ 3つのファイルは必ず同じフォルダに置いてください


================================================================
  毎日の使い方
================================================================

1. KINKAN.bat をダブルクリック
   → 2つの黒いウィンドウが開きます（正常です・最小化でOK）
   → Chromeが自動で起動します

2. 「Googleログイン（準備完了）」ボタンをクリック
   → [EMAIL_ADDRESS] でログイン

3. 「▶ 出勤」ボタンを押して作業開始
   → ポモドーロタイマーが自動でスタートします

4. 業務終了時に「■ 退勤」ボタンを押す

5. 使い終わったら黒いウィンドウを閉じる
   （サーバーが停止します）


================================================================
  機能一覧
================================================================

【出退勤】
  - 出勤ボタン：出勤時刻を記録
  - 退勤ボタン：退勤時刻を記録・勤務時間を自動計算
  - 勤務時間は30秒ごとに自動保存されます

【同日複数回出勤】
  - 退勤後に同じ日に再度「▶ 出勤」ボタンを押せます
  - 退勤〜再出勤の間の時間は自動で休憩として加算されます
  - 例）09:00出勤→12:00退勤→13:00出勤→18:00退勤
         → 勤務時間8時間、休憩1時間として記録
  - 退勤後のステータスは「離席中（再出勤できます）」と表示

【ポモドーロタイマー】
  - 25分作業 → 5分休憩を繰り返す集中タイマー
  - 「出勤時に自動スタート」はデフォルトでオン
  - チェックを外すと手動スタートになります
  - 設定はブラウザに記憶されます

【休憩管理】
  - 任意のタイミングで休憩の開始・終了を記録
  - 休憩時間は勤務時間から自動で除外
  - 同日再出勤時の離席時間も自動で休憩に加算

【月次レポート（上部タブ）】
  - 月の全日程を一覧表示（未勤務日は空白）
  - < > ボタンで過去・未来の月に移動できます
  - 行をクリックすると記録を編集・追加できます
  - 手動で編集した記録には「手動」バッジが表示されます
  - 棒グラフで月の勤務時間を視覚的に確認できます

【Google Sheetsへの自動同期】
  - 記録のたびに自動でGoogle Driveへ保存
  - 勤務中も30秒ごとに自動保存
  - 月ごとにシートが分かれます
  - 複数PCで同じGoogleアカウントを使えばデータ共有可能


================================================================
  2つの黒いウィンドウについて
================================================================

  ■ 「続行するには何かキーを押してください」ウィンドウ
    → アプリを使う間は最小化しておいてください
    → 閉じるとサーバーが停止してアプリが使えなくなります

  ■ 「Serving HTTP on :: port 8080」ウィンドウ
    → Pythonのサーバーが動作中のウィンドウです
    → 同様に最小化して使用してください

  ※ 終了するときはどちらかのウィンドウを閉じてください


================================================================
  別のPCで使う場合
================================================================

1. このフォルダごとUSBメモリ等でコピー
2. コピー先のPCにPythonがインストールされているか確認
     → コマンドプロンプトで「python --version」と入力
     → 表示されない場合は https://python.org からインストール
3. KINKAN.bat をダブルクリックして起動
4. 同じGoogleアカウントでログインすれば同じデータを使用可能


================================================================
  データのリセット方法
================================================================

テスト中のデータを消して本番環境に切り替えたい場合は
以下の手順でリセットしてください。

【手順1】ブラウザのlocalStorageをクリア

  1. kinkan.html を開いた状態で F12
  2. 「Application」タブをクリック
  3. 左メニュー「Local Storage」
     → 「http://localhost:8080」をクリック
  4. 「http://localhost:8080」を右クリック → 「Clear」

【手順2】Google Sheetsのスプレッドシートを削除

  1. https://drive.google.com を開く
  2. 「KINKAN 勤怠記録 2026」を右クリック
     → 「ゴミ箱に移動」
  3. ゴミ箱を空にする

【手順3】アプリを再起動

  1. KINKAN.bat をダブルクリック
  2. Googleログイン後、「+ 新規作成」をクリック
  3. 新しいスプレッドシートが作成されてリセット完了


================================================================
  トラブルシューティング
================================================================

【アプリが開かない】
  → Chromeがインストールされているか確認
  → コマンドプロンプトで「python --version」を確認

【「準備完了」が表示されない】
  → インターネット接続を確認
  → F5キーでページを再読み込み

【ログインできない】
  → kazu80330@gmail.com でログインしているか確認
  → インターネット接続を確認

【データがGoogle Sheetsに反映されない】
  → バナーの同期ドットが緑になっているか確認
  → 一度ログアウトして再ログイン

【勤務時間が表示されない】
  → 退勤ボタンを押すと確定されます
  → 勤務中は30秒ごとに自動保存されます


================================================================
  データの保存場所
================================================================

  ブラウザ内（localStorage）: 一時的なデータの保存
  Google Drive               : 「KINKAN 勤怠記録 2026」
                               スプレッドシートに自動保存
